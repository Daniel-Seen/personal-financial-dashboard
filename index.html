Connect Cloud Sync
Sync data between your iPhone and Mac using JSONBin.io (free).

1. Go to jsonbin.io on any browser
2. Create a free account
3. Click Bins → Create a Bin, paste {} as content
4. Copy your Master Key (Account → API Keys)
5. Copy the Bin ID from the URL
